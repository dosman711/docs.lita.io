Overview

Lita's plugin system is used to connect with different chat services and to add new runtime behavior. This page provides details for plugin authors to create their own adapters and handlers. If you're just learning about Lita, you may want to check out the getting started page first.

Plugins are written in Ruby, so you'll need some familiarity with Ruby and its ecosystem of tools to write and publish a plugin. To test plugins, Lita provides some useful additions for the RSpec testing framework. It's not necessary to use them, but familiarity with RSpec will be beneficial in testing your plugins.
